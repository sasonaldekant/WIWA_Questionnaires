# Feature Implementation Loop

## 1. Plan (FullStack Developer)
Trigger the **FullStack Developer** skill.
- **Context**: Focus on a single feature / bug.
- **Action**: Outline the code changes in a scratchpad or thought process.

## 2. Code (FullStack Developer)
Trigger the **FullStack Developer** skill.
- **Action**: Apply the code changes (Backend & Frontend).

## 3. Verify (QA Engineer)
Trigger the **QA Engineer** skill.
- **Action**: Quick check of the specific feature.
- **Output**: Pass/Fail.
